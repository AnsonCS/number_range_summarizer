# Number Range Summarizer

This project implements the **NumberRangeSummarizer** interface, which takes a comma-delimited list of integers and summarizes consecutive numbers into ranges.

**Input:**
"1,3,6,7,8,12,13,14,15,21,22,23,24,31"


**Output:**
"1, 3, 6-8, 12-15, 21-24, 31"

## Requirements

- **Language:** Java 8 or higher  
- **Build Tool:** Maven
- **Testing Framework:** JUnit 5  
- **Version Control:** GitHub
